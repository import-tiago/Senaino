# Contributing

Contributions are welcome, especially improvements that make the board easier to fabricate, assemble, test or teach.

## Suggested Flow

1. Open an issue describing the proposed change.
2. Fork the repository.
3. Create a focused branch.
4. Update documentation together with hardware changes.
5. Submit a pull request with screenshots, Gerber previews or test notes when relevant.

## Hardware Contribution Notes

- Keep the educational assembly workflow in mind.
- Prefer components that are available in Brazil.
- Document any footprint, BOM or fabrication change.
- Include clear notes when a change affects the FT232 or CH340 variants differently.
