# Licensing

Senaino is distributed as an open hardware project with separate licenses for hardware design material and educational/documentation material.

## Hardware License

The hardware design sources and manufacturing files are licensed under the CERN Open Hardware Licence Version 2 - Strongly Reciprocal.

SPDX identifier: `CERN-OHL-S-2.0`

This applies to:

- `hardware/eagle/`
- `hardware/gerbers/`
- `hardware/exports/`
- `hardware/bom/`

The CERN-OHL-S-2.0 license is a hardware-focused reciprocal license. It is more appropriate for PCB design files than using only a documentation/content license.
