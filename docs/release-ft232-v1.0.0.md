# Senaino FT232 v1.0.0 Release Notes

## Summary

This release archives the FT232 version of Senaino, an educational Arduino UNO-inspired open hardware board designed for soldering, electronics and embedded programming training.

## Included Materials

- Eagle schematic and board files.
- Open Gerber fabrication files.
- ZIP fabrication package.
- Schematic PDF and image exports.
- Bill of materials in XLSX and CSV formats.
- Assembly guide.
- Testing checklist.
- Driver and bootloader notes.
- Citation metadata for DOI-based release.
- License overview for hardware and documentation assets.

## Variant

| Field | Value |
|---|---|
| Variant | FT232 |
| USB-serial IC | FT232RL |
| Microcontroller | ATmega328P-PU |
| Main clock | 16 MHz |
| Release tag | `ft232-v1.0.0` |
